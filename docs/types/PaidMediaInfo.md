# Type: PaidMediaInfo

**Category**: Available types

**Description**: Describes the paid media added to a message.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#paidmediainfo)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| star_count | Integer | Yes | The number of Telegram Stars that must be paid to buy access to the media |
| paid_media | Array of PaidMedia | Yes | Information about the paid media |
