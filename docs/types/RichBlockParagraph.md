# Type: RichBlockParagraph

**Category**: Rich messages

**Description**: A text paragraph, corresponding to the HTML tag <p>.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#richblockparagraph)

**Subtype of**: RichBlock

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| type | String | Yes | Type of the block, always "paragraph" |
| text | RichText | Yes | Text of the block |
