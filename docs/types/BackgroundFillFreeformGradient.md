# Type: BackgroundFillFreeformGradient

**Category**: Available types

**Description**: The background is a freeform gradient that rotates after every message in the chat.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#backgroundfillfreeformgradient)

**Subtype of**: BackgroundFill

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| type | String | Yes | Type of the background fill, always "freeform_gradient" |
| colors | Array of Integer | Yes | A list of the 3 or 4 base colors that are used to generate the freeform gradient in the RGB24 format |
