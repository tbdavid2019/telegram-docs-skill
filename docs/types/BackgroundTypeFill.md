# Type: BackgroundTypeFill

**Category**: Available types

**Description**: The background is automatically filled based on the selected colors.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#backgroundtypefill)

**Subtype of**: BackgroundType

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| type | String | Yes | Type of the background, always "fill" |
| fill | BackgroundFill | Yes | The background fill |
| dark_theme_dimming | Integer | Yes | Dimming of the background in dark themes, as a percentage; 0-100 |
