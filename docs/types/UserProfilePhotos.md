# Type: UserProfilePhotos

**Category**: Available types

**Description**: This object represent a user's profile pictures.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#userprofilephotos)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| total_count | Integer | Yes | Total number of profile pictures the target user has |
| photos | Array of Array of PhotoSize | Yes | Requested profile pictures (in up to 4 sizes each) |
