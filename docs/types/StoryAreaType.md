# Type: StoryAreaType

**Category**: Available types

**Description**: Describes the type of a clickable area on a story. Currently, it can be one of
- StoryAreaTypeLocation
- StoryAreaTypeSuggestedReaction
- StoryAreaTypeLink
- StoryAreaTypeWeather
- StoryAreaTypeUniqueGift

**Link**: [Official Documentation](https://core.telegram.org/bots/api#storyareatype)

**Subtypes**: StoryAreaTypeLocation, StoryAreaTypeSuggestedReaction, StoryAreaTypeLink, StoryAreaTypeWeather, StoryAreaTypeUniqueGift
