# Type: SentWebAppMessage

**Category**: Available types

**Description**: Describes an inline message sent by a Web App on behalf of a user.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#sentwebappmessage)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| inline_message_id | String | Optional | Optional. Identifier of the sent inline message. Available only if there is an inline keyboard attached to the message. |
