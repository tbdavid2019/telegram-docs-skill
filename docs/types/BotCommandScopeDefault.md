# Type: BotCommandScopeDefault

**Category**: Available types

**Description**: Represents the default scope of bot commands. Default commands are used if no commands with a narrower scope are specified for the user.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#botcommandscopedefault)

**Subtype of**: BotCommandScope

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| type | String | Yes | Scope type, must be default |
