# Type: ChatBoostSourcePremium

**Category**: Available types

**Description**: The boost was obtained by subscribing to Telegram Premium or by gifting a Telegram Premium subscription to another user.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#chatboostsourcepremium)

**Subtype of**: ChatBoostSource

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| source | String | Yes | Source of the boost, always "premium" |
| user | User | Yes | User that boosted the chat |
