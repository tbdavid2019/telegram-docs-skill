# Type: BackgroundTypeChatTheme

**Category**: Available types

**Description**: The background is taken directly from a built-in chat theme.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#backgroundtypechattheme)

**Subtype of**: BackgroundType

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| type | String | Yes | Type of the background, always "chat_theme" |
| theme_name | String | Yes | Name of the chat theme, which is usually an emoji |
