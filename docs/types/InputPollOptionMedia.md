# Type: InputPollOptionMedia

**Category**: Available types

**Description**: This object represents the content of a poll option to be sent. It should be one of
- InputMediaAnimation
- InputMediaLink
- InputMediaLivePhoto
- InputMediaLocation
- InputMediaPhoto
- InputMediaSticker
- InputMediaVenue
- InputMediaVideo

**Link**: [Official Documentation](https://core.telegram.org/bots/api#inputpolloptionmedia)

**Subtypes**: InputMediaAnimation, InputMediaLink, InputMediaLivePhoto, InputMediaLocation, InputMediaPhoto, InputMediaSticker, InputMediaVenue, InputMediaVideo
