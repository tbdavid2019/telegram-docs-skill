# Type: StoryAreaTypeUniqueGift

**Category**: Available types

**Description**: Describes a story area pointing to a unique gift. Currently, a story can have at most 1 unique gift area.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#storyareatypeuniquegift)

**Subtype of**: StoryAreaType

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| type | String | Yes | Type of the area, always "unique_gift" |
| name | String | Yes | Unique name of the gift |
