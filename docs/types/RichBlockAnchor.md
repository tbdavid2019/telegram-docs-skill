# Type: RichBlockAnchor

**Category**: Rich messages

**Description**: A block with an anchor, corresponding to the HTML tag <a> with the attribute name.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#richblockanchor)

**Subtype of**: RichBlock

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| type | String | Yes | Type of the block, always "anchor" |
| name | String | Yes | The name of the anchor |
