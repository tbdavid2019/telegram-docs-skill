# Type: PreparedInlineMessage

**Category**: Available types

**Description**: Describes an inline message to be sent by a user of a Mini App.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#preparedinlinemessage)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| id | String | Yes | Unique identifier of the prepared message |
| expiration_date | Integer | Yes | Expiration date of the prepared message, in Unix time. Expired prepared messages can no longer be used |
