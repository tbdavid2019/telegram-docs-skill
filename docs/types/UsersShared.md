# Type: UsersShared

**Category**: Available types

**Description**: This object contains information about the users whose identifiers were shared with the bot using a KeyboardButtonRequestUsers button.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#usersshared)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| request_id | Integer | Yes | Identifier of the request |
| users | Array of SharedUser | Yes | Information about users shared with the bot. |
