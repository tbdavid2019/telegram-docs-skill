# Type: UserProfileAudios

**Category**: Available types

**Description**: This object represents the audios displayed on a user's profile.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#userprofileaudios)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| total_count | Integer | Yes | Total number of profile audios for the target user |
| audios | Array of Audio | Yes | Requested profile audios |
