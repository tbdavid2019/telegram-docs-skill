# Type: CopyTextButton

**Category**: Available types

**Description**: This object represents an inline keyboard button that copies specified text to the clipboard.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#copytextbutton)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| text | String | Yes | The text to be copied to the clipboard; 1-256 characters |
