# Type: PaidMediaPreview

**Category**: Available types

**Description**: The paid media isn't available before the payment.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#paidmediapreview)

**Subtype of**: PaidMedia

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| type | String | Yes | Type of the paid media, always "preview" |
| width | Integer | Optional | Optional. Media width as defined by the sender |
| height | Integer | Optional | Optional. Media height as defined by the sender |
| duration | Integer | Optional | Optional. Duration of the media in seconds as defined by the sender |
