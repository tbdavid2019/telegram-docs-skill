# Type: Dice

**Category**: Available types

**Description**: This object represents an animated emoji that displays a random value.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#dice)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| emoji | String | Yes | Emoji on which the dice throw animation is based |
| value | Integer | Yes | Value of the dice, 1-6 for "🎲", "🎯" and "🎳" base emoji, 1-5 for "🏀" and "⚽" base emoji, 1-64 for "🎰" base emoji |
