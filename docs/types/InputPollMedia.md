# Type: InputPollMedia

**Category**: Available types

**Description**: This object represents the content of a poll description or a quiz explanation to be sent. It should be one of
- InputMediaAnimation
- InputMediaAudio
- InputMediaDocument
- InputMediaLivePhoto
- InputMediaLocation
- InputMediaPhoto
- InputMediaVenue
- InputMediaVideo

**Link**: [Official Documentation](https://core.telegram.org/bots/api#inputpollmedia)

**Subtypes**: InputMediaAnimation, InputMediaAudio, InputMediaDocument, InputMediaLivePhoto, InputMediaLocation, InputMediaPhoto, InputMediaVenue, InputMediaVideo
