# Type: OwnedGift

**Category**: Available types

**Description**: This object describes a gift received and owned by a user or a chat. Currently, it can be one of
- OwnedGiftRegular
- OwnedGiftUnique

**Link**: [Official Documentation](https://core.telegram.org/bots/api#ownedgift)

**Subtypes**: OwnedGiftRegular, OwnedGiftUnique
