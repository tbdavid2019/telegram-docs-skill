# Type: UniqueGiftBackdropColors

**Category**: Available types

**Description**: This object describes the colors of the backdrop of a unique gift.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#uniquegiftbackdropcolors)

### Fields

| Field | Type | Required | Description |
|---|---|---|---|
| center_color | Integer | Yes | The color in the center of the backdrop in RGB format |
| edge_color | Integer | Yes | The color on the edges of the backdrop in RGB format |
| symbol_color | Integer | Yes | The color to be applied to the symbol in RGB format |
| text_color | Integer | Yes | The color for the text on the backdrop in RGB format |
