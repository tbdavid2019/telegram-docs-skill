# Method: getChatAdministrators

**Category**: Available methods

**Description**: Use this method to get a list of administrators in a chat, which aren't bots. Returns an Array of ChatMember objects.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#getchatadministrators)

**Returns**: Array of ChatMember

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| chat_id | Integer or String | Yes | Unique identifier for the target chat or username of the target supergroup or channel (in the format @channelusername) |
