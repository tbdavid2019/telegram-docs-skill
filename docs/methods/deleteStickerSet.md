# Method: deleteStickerSet

**Category**: Stickers

**Description**: Use this method to delete a sticker set that was created by the bot. Returns True on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#deletestickerset)

**Returns**: Boolean

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| name | String | Yes | Sticker set name |
