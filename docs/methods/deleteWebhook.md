# Method: deleteWebhook

**Category**: Getting updates

**Description**: Use this method to remove webhook integration if you decide to switch back to getUpdates. Returns True on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#deletewebhook)

**Returns**: Boolean

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| drop_pending_updates | Boolean | Optional | Pass True to drop all pending updates |
