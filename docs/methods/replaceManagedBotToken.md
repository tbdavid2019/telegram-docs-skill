# Method: replaceManagedBotToken

**Category**: Available methods

**Description**: Use this method to revoke the current token of a managed bot and generate a new one. Returns the new token as String on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#replacemanagedbottoken)

**Returns**: String

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| user_id | Integer | Yes | User identifier of the managed bot whose token will be replaced |
