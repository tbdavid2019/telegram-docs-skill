# Method: setMyProfilePhoto

**Category**: Available methods

**Description**: Changes the profile photo of the bot. Returns True on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#setmyprofilephoto)

**Returns**: Boolean

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| photo | InputProfilePhoto | Yes | The new profile photo to set |
