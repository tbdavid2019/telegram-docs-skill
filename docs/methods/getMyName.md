# Method: getMyName

**Category**: Available methods

**Description**: Use this method to get the current bot name for the given user language. Returns BotName on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#getmyname)

**Returns**: BotName

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| language_code | String | Optional | A two-letter ISO 639-1 language code or an empty string |
