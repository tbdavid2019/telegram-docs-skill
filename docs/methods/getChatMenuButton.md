# Method: getChatMenuButton

**Category**: Available methods

**Description**: Use this method to get the current value of the bot's menu button in a private chat, or the default menu button. Returns MenuButton on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#getchatmenubutton)

**Returns**: MenuButton

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| chat_id | Integer | Optional | Unique identifier for the target private chat. If not specified, default bot's menu button will be returned |
