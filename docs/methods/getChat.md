# Method: getChat

**Category**: Available methods

**Description**: Use this method to get up-to-date information about the chat. Returns a ChatFullInfo object on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#getchat)

**Returns**: ChatFullInfo

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| chat_id | Integer or String | Yes | Unique identifier for the target chat or username of the target supergroup or channel (in the format @channelusername) |
