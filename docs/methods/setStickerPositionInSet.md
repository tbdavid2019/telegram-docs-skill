# Method: setStickerPositionInSet

**Category**: Stickers

**Description**: Use this method to move a sticker in a set created by the bot to a specific position. Returns True on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#setstickerpositioninset)

**Returns**: Boolean

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| sticker | String | Yes | File identifier of the sticker |
| position | Integer | Yes | New sticker position in the set, zero-based |
