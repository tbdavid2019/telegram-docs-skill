# Method: getMyDescription

**Category**: Available methods

**Description**: Use this method to get the current bot description for the given user language. Returns BotDescription on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#getmydescription)

**Returns**: BotDescription

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| language_code | String | Optional | A two-letter ISO 639-1 language code or an empty string |
