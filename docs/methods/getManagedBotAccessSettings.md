# Method: getManagedBotAccessSettings

**Category**: Available methods

**Description**: Use this method to get the access settings of a managed bot. Returns a BotAccessSettings object on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#getmanagedbotaccesssettings)

**Returns**: BotAccessSettings

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| user_id | Integer | Yes | User identifier of the managed bot whose access settings will be returned |
