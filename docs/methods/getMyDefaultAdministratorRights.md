# Method: getMyDefaultAdministratorRights

**Category**: Available methods

**Description**: Use this method to get the current default administrator rights of the bot. Returns ChatAdministratorRights on success.

**Link**: [Official Documentation](https://core.telegram.org/bots/api#getmydefaultadministratorrights)

**Returns**: ChatAdministratorRights

### Parameters

| Field | Type | Required | Description |
|---|---|---|---|
| for_channels | Boolean | Optional | Pass True to get default administrator rights of the bot in channels. Otherwise, default administrator rights of the bot for groups and supergroups will be returned. |
